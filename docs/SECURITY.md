# Security Policy

## Supported Versions
- v1.x (latest release)

## Reporting a Vulnerability
If you discover a security issue, please email anjumy.ai@gmail.com with:
- Description of the vulnerability
- Steps to reproduce / PoC
- Impact assessment

We aim to acknowledge reports within 3 business days and provide a fix timeline as soon as possible.

## Disclosure
Please do not disclose vulnerabilities publicly until we release a fix.
