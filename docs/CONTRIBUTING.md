# Contributing

Thanks for your interest in contributing! Follow these steps:

1. Fork the repository and create a feature branch (git checkout -b feat/xyz).
2. Install dependencies with pip install -r requirements.txt.
3. Run python app.py to test changes locally.
4. Submit a pull request with a clear description, screenshots, and test notes.

## Guidelines
- Follow semantic versioning.
- Keep commits focused and descriptive.
- Ensure UI changes work in both light and dark themes.

## Reporting Issues
Open an issue with steps to reproduce, expected vs actual behavior, and environment details.
